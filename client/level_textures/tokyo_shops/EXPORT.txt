Automated (after redesign-tokyo-phase1 created ShopFront_*_recess):

  python3 tools/oyabaunctl.py fix-tokyo-shopfront-materials
  python3 tools/oyabaunctl.py apply-tokyo-shop-textures
  python3 tools/oyabaunctl.py export-world --force-all

fix-tokyo-shopfront-materials: recess -> OYA_Recess, awnings -> OYA_Awning* (not brick/trim on thin quads).

This adds OyabaunShopPanels planes with OYA_ShopFace_* materials; enhance skips those materials.

Manual Blender (optional):
  1. Image → Open, pick PNG, set Color Space to sRGB.
  2. Pack image (File → External Data → Pack Resources) before export.
  3. Apply to shop face planes: UV scale for ~3–5m wide facade; Materials → Principled BSDF
     Base Color ← Image Texture, Interpolation Nearest (pixel art).
  4. Emissive on window/neon areas if you split quads or use a second material pass.
  5. Re-export GLB: tools/blender_export_gltf_oyabaun.py or oyabaunctl export-world; then wasm-pack if embedding.
